# 포함된 내용

- 기어 메뉴 추가  
- 설정파일에 값 저장  
- 단축키 설정  
- GUI 기본 작성. 윈도우 리사이즈시 밖으로 나가버리는거 방지 포함  
- 코루틴  
- 하모니  
- OnSceneLoaded 기본 사용법.  scene.buildIndex는 쓰지 말자 제발.  

바로 컴파일 할수 있게 BepInPluginSample/lib/ 폴더 안에 dll 들어 있음.  
게임 버전에 맞게 기존 게임 폴더에서 가져다 쓸것.
